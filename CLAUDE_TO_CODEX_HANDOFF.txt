IMPLEMENTED CHANGES

Onboarding
- src/onboarding/OnboardingFlow.tsx now has an interactive "Choose your defaults" panel on the first step.
- The picker includes Instagram, YouTube, X, TikTok, Facebook, and Snapchat.
- Site chips are accessible buttons with selected state and 44px+ touch targets.
- src/onboarding/steps.ts copy is more inviting and now honestly explains that Screen Time control requires Apple's restricted Family Controls entitlement, so Stillpoint Personal Edition uses Shortcuts redirects instead.
- The existing guard that onboarding copy never says "block" still passes.

Breathe shortcut
- Added src/screens/BreatheScreen.tsx.
- It is a calm shortcut landing screen with a tappable breathing orb cycling Inhale -> Hold -> Exhale -> Rest.
- Added Breathe as a hidden bottom-tab screen in src/nav/RootNavigator.tsx.
- Added stillpoint://breathe in src/nav/linking.ts.

Social tab
- src/screens/SocialScreen.tsx has been redesigned into a warm SocialLite-inspired surface:
  - Title: "Stillpoint Social"
  - Subtitle: "Focus forward"
  - Rounded translucent site cards.
  - App-style badges for Instagram, YouTube, X, TikTok, Facebook, Snapchat.
  - BETA pills for the newer/less certain sites.
  - Per-site settings buttons and chevrons.
  - "Add apps to Home Screen" card.
  - Bottom dots and share/help row.
- This still uses the existing onOpenSite callback, so friction gates remain intact.

Sanitized browser
- src/screens/BrowserScreen.tsx now feels like an integrated sanitized app shell:
  - Top status strip with elapsed time, "0 ads", "0 suggested".
  - Blue "Block these" action, matching the user's SocialLite screenshots.
  - Full-screen "Applying your preferences" transition overlay.
  - Floating menu button.
  - Expandable floating toolbar with back, home, refresh, forward, report, settings.
  - WebView still has sharedCookiesEnabled and domStorageEnabled to preserve logins between visits.
- No extra native dependencies were added.

Sanitizer rules
- src/sanitizer/types.ts now supports:
  instagram, youtube, x, tiktok, facebook, snapchat.
- src/sanitizer/rules.ts adds first-pass Facebook and Snapchat rules.
- YouTube selectors were expanded slightly to hide more recommendation surfaces.
- Remember: site CSS selectors will drift and need upkeep.
