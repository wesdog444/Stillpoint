Stillpoint handoff - Strict Purpose Mode sanitizer
Date: 2026-05-21
Current branch: codex/strict-sanitizer

SESSION SUMMARY
- User asked for a stricter sanitizer for all supported apps, specifically including YouTube and TikTok.
- I recommended a Strict Purpose Mode pass:
  - stronger CSS selectors,
  - text-label blocking for labels like Reels, Shorts, For You, Discover, Suggested, Recommended, Trending,
  - repeated cleanup for dynamic pages,
  - autoplay/video disabling.
- User approved: "Yes lets do it!"
- Implemented the strict sanitizer changes test-first.

WHAT CHANGED

Rule model
- src/sanitizer/types.ts:
  - Added `textBlocklist: string[]`.
  - Added optional `disableVideoAutoplay?: boolean`.

Injector
- src/sanitizer/injection.ts:
  - Builds a `textBlocklist` array into the injected WebView script.
  - Adds `hideByStillpointLabel()` to scan visible labels and hide the nearest clickable/card/nav surface.
  - Adds `disableStillpointVideos()` to pause videos, mute them, remove autoplay, and set autoplay false.
  - Calls these inside the existing repeated cleanup loop so dynamic content gets re-sanitized after load.

Rules
- src/sanitizer/rules.ts:
  - Instagram: stricter Reels/Explore/Suggested/feed selectors plus text labels.
  - YouTube: stricter Shorts/recommendations/watch-next/comments/end-screen/autoplay selectors plus labels and video disabling.
  - X: stricter Explore/Trending/Who to follow/sidebar selectors plus labels and video disabling.
  - TikTok: stricter For You/Explore/Discover/feed/suggested selectors plus labels and video disabling.
  - Facebook: stricter Reels/Watch/Stories/Marketplace/feed selectors plus labels and video disabling.
  - Snapchat: stricter Spotlight/Discover/Suggested selectors plus labels and video disabling.

Tests
- src/sanitizer/__tests__/rules.test.ts:
  - Verifies every rule has textBlocklist.
  - Verifies YouTube strict selectors/text labels/autoplay disabling.
  - Verifies TikTok strict selectors/text labels/autoplay disabling.
- src/sanitizer/__tests__/injection.test.ts:
  - Verifies text label blocking is injected.
  - Verifies video disabling is injected.
