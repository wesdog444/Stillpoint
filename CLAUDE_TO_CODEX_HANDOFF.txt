=== STEP-BY-STEP: WHAT THE OWNER MUST DO (Plan 8 tasks 1 & 5) ===
These cannot be done from Windows by an agent — they need the owner's GitHub account.
1. Create a private GitHub repo named `stillpoint` (https://github.com/new), OR with the
   gh CLI: `gh repo create stillpoint --private --source . --remote origin --push`.
2. If not pushed by gh: `git remote add origin https://github.com/<owner>/stillpoint.git`
   then `git push -u origin master`.
3. The push triggers the "Build unsigned IPA" workflow (GitHub repo -> Actions tab).
4. When it finishes (~15-30 min), download the `Stillpoint-unsigned-ipa` artifact.
5. Follow docs/SIDELOAD.md to sign + install with Sideloadly.
If the CI build fails: Plan 8 Task 5 Step 3 lists the likely fixes (scheme name, build-
product path, transient pod errors) — iterate on .github/workflows/build-ipa.yml.
